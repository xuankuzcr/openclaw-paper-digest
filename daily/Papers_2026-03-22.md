## 🧾 今日结论（先看这5条）
- **趋势判断：生成式模型正从“会生成”走向“会理解空间”**。`Generation Models Know Space` 这类工作说明2D大模型里隐含的3D先验可被系统挖掘，预示未来机器人感知可能减少对重标注3D数据和昂贵几何管线的依赖。  
- **方法走向：预训练先验 + 轻量强化微调成为新范式**。`From Prior to Pro` 强调分布收缩式RL微调，本质是在“守住先验能力边界”下做技能专精，能缓解RL常见的灾难性偏移，适合从通用模型到任务专家的过渡。  
- **表征升级：视频自监督向稠密特征与世界模型深耦合**。`V-JEPA 2.1` 这类进展意味着机器人可从被动看视频中学到更可控、可迁移的时空表征，为预测控制、长时序规划和跨场景泛化打基础。  
- **落地可行性：仿真到现实链路正在变短，但仍卡在闭环验证**。4DGS+Diffusion模拟器与世界模型结合有望显著提升数据效率和策略迭代速度，不过真实部署仍需解决接触动力学误差、传感延迟和安全约束未建模问题。  
- **风险与机会：机会在“低成本通用能力”，风险在“隐式先验不可解释”**。若3D/时空知识主要是隐式涌现，系统在OOD场景下可能出现自信但错误的决策；建议同步投入不确定性估计、可解释诊断与安全监控，才能把研究红利转成工业可靠性。

## 📚 分类论文清单（便于速读）

### 🧠 World Models（1）

#### ⭐ V-JEPA 2.1: Unlocking Dense Features in Video Self-Supervised Learning
- **主题**：World Models
- **论文链接**：
- **辅助图示**：https://cdn-thumbnails.hf-mirror.com/social-thumbnails/papers/2603.14482.png
![V-JEPA 2.1: Unlocking Dense Features in Video Self-Supervised Learning](https://cdn-thumbnails.hf-mirror.com/social-thumbnails/papers/2603.14482.png)
- **这篇在做什么**：可以把V-JEPA 2.1理解为：让模型在看见部分画面时，不仅猜整体语义，还要在空间位置和时间变化上都“对得上”。它不只在最后一层学表示，而是在中间多层都加自监督约束，因此特征更稳、更细。最终得到的表示既能做全局识别，也能支持抓取、导航、深度等需要局部几何与时序一致性的任务。
- **核心方法**：输入侧同时支持图像与视频，通过多模态tokenizer把不同来源转成可统一处理的token表示；训练时采用掩码预测框架，但可见与被遮挡token都参与稠密预测损失。模型在编码器多个中间层施加深层自监督，让各层都学习可用语义与时空结构。整体流程是大规模自监督预训练后，将表征迁移到下游任务进行线性评估或任务头微调/推理。
- **和以前有什么不同**：相较早期JEPA或常见掩码重建方法，本工作强调“稠密预测+可见token同训”，减少只对遮挡区域学习导致的局部偏置。它还把自监督从末层扩展到层级式中间层监督，提升特征可迁移性与训练稳定性。再加上图像-视频统一token化与规模扩展，形成了比单一视频SSL更通用的表征学习框架。
- **你能学到什么**：阅读时可重点拆成四个可复用设计：损失覆盖范围、监督注入深度、输入统一表示、以及规模化策略，这比只看榜单更有迁移价值。实践上可先在现有视频骨干中加入中间层监督与可见token损失，通常就能观察到稠密任务收益。另一个启发是评估应同时看全局识别与具身任务，避免表征只在单一指标上优化。
- **关键洞见**：论文的关键洞见是：高质量世界建模表征不应在“全局语义”和“局部时空细节”之间二选一，而要通过目标函数与监督位置共同设计来兼得。把训练信号从“只预测被遮挡部分”扩展为“全token稠密对齐”，并在多层持续约束，可显著增强空间结构、语义一致性和时间连贯性。统一图像与视频训练则进一步提升了表示的通用性与下游覆盖面。

### 🎮 4DGS + Diffusion Simulator（1）

#### ⭐ From Prior to Pro: Efficient Skill Mastery via Distribution Contractive RL Finetuning
- **主题**：4DGS + Diffusion Simulator
- **论文链接**：
- **辅助图示**：https://cdn-thumbnails.hf-mirror.com/social-thumbnails/papers/2603.10263.png
![From Prior to Pro: Efficient Skill Mastery via Distribution Contractive RL Finetuning](https://cdn-thumbnails.hf-mirror.com/social-thumbnails/papers/2603.10263.png)
- **这篇在做什么**：可把DICE-RL理解为：先训练一个“会很多但不精”的行为先验，再用在线奖励把它往“成功动作更密集”的区域压缩。它不是完全推翻原策略，而是在先验附近做有约束的强化提升，所以更稳、更不容易学崩。最终目标是用更少交互，把通用机器人策略打磨成任务专家。
- **核心方法**：输入与表示上，方法以高维像素观测为条件，策略可由扩散或流模型参数化，先通过预训练获得广覆盖行为分布。微调阶段采用残差式离策略RL：在先验动作基础上学习增量修正，并配合选择性行为正则，避免策略偏离可行先验过远。推理时结合价值引导的动作选择，在候选动作中偏向高价值样本，实现“保留先验多样性+强化成功模式”的分布收缩。
- **和以前有什么不同**：相较从零开始的在线RL，它利用生成式先验显著降低探索难度，把学习重点放在“筛优与放大”而非“全面重建”。相较仅做行为克隆或纯离线微调，它引入在线价值信号持续纠偏，能突破示范上限并适应真实交互反馈。相较常规策略梯度微调，残差离策略+选择性正则的组合更强调稳定与样本效率。
- **你能学到什么**：阅读时可重点抓住“distribution contraction”这一本体：RL在此被定义为对行为分布的收缩算子，而不是单纯参数更新。第二个要点是三件套协同：生成式先验提供覆盖、残差离策略提供可控改进、价值引导负责落地决策。复现层面需特别关注正则强度与价值估计质量，它们通常决定稳定性与上限。
- **关键洞见**：核心洞见是：当先验已覆盖足够行为模式时，最有效的RL不是扩大搜索，而是把概率质量向高成功轨迹系统性迁移。通过“受约束的偏移”替代“激进重写”，可以同时获得高性能、稳定训练和较好样本效率。这为“基础机器人策略→任务专家策略”的通用途径提供了清晰范式。

### 📌 Other（1）

#### ⭐ Generation Models Know Space: Unleashing Implicit 3D Priors for Scene Understanding
- **主题**：Other
- **论文链接**：
- **辅助图示**：https://cdn-thumbnails.hf-mirror.com/social-thumbnails/papers/2603.19235.png
![Generation Models Know Space: Unleashing Implicit 3D Priors for Scene Understanding](https://cdn-thumbnails.hf-mirror.com/social-thumbnails/papers/2603.19235.png)
- **这篇在做什么**：作者认为，能生成时序一致视频的扩散模型，已经在训练中隐式学会了物体结构、深度关系和运动规律。VEGA-3D把这种能力当作“潜在世界模拟器”，提取中间特征再喂给MLLM，从而让模型在看图问答、空间推理和操作理解上更懂几何与物理。
- **核心方法**：输入侧同时使用语义信息与视频扩散模型的时空潜特征：前者提供类别与语义，后者提供隐式3D线索。关键模块是从扩散过程的中间噪声层抽取特征，并通过token级自适应门控融合，把几何线索按需注入语言-视觉表示。训练/推理上无需显式3D监督，作为即插即用增强器接入现有MLLM，在下游任务中利用融合表征完成理解与决策。
- **和以前有什么不同**：相较依赖点云、深度或NeRF等显式3D输入的方法，VEGA-3D转向利用生成模型内部先验，减少对稀缺3D标注和复杂重建流程的依赖。相较仅靠2D视觉语义增强的方案，它引入时空一致的潜在几何证据，因此在细粒度空间关系与物理动态判断上更有优势。
- **你能学到什么**：可借鉴点一是“生成即世界模型”的思路：把生成网络中间状态视作可复用的知识库，而非只看最终生成结果。可借鉴点二是融合策略要细粒度、可调节，token级门控比粗拼接更能抑制噪声并保留任务相关几何信息；实践中可先做冻结主干、轻量适配。
- **关键洞见**：核心洞见是：大规模视频生成模型为了保证跨帧一致性，已被迫学习到可迁移的3D结构与物理规律，这些先验能直接服务于理解任务。换言之，空间智能不一定要靠显式3D监督“教出来”，也可以从生成模型里“提炼出来”。
