## 📚 分类论文清单（便于速读）

### 🧠 World Models（1）

#### ⭐ V-JEPA 2.1: Unlocking Dense Features in Video Self-Supervised Learning
- **主题**：World Models
- **论文链接**：
- **辅助图示**：https://cdn-thumbnails.hf-mirror.com/social-thumbnails/papers/2603.14482.png
![V-JEPA 2.1: Unlocking Dense Features in Video Self-Supervised Learning](https://cdn-thumbnails.hf-mirror.com/social-thumbnails/papers/2603.14482.png)
- **这篇在做什么**：可以把 V-JEPA 2.1 理解为：让模型在看见部分画面时，不只猜整体含义，还要在空间位置和时间变化上“对齐”被遮挡内容，从而学到可用于精细决策的特征。它通过分层监督和统一图像/视频训练，让同一套表示既懂场景语义，也保留像素邻域与运动线索，因此在多类视觉与机器人任务上更稳。
- **核心方法**：输入是图像或视频token，经多模态tokenizer映射到统一表示空间；训练时采用mask预测范式，但监督信号同时来自可见与被遮挡token，以强化空间和时间定位能力。主干编码器的多个中间层都施加自监督目标（深层级监督），而非只在顶层学习。整体流程是统一预训练得到通用表征，再以线性探针或任务头进行下游推理，兼顾全局理解与密集特征输出。
- **和以前有什么不同**：相较早期JEPA或常见掩码重建方法，V-JEPA 2.1 的关键增量在于“密集预测损失+中间层分级监督+图像视频统一token化+可扩展训练”的组合，而不是单点改动。它强调时空结构一致性的显式学习，减少仅依赖顶层语义导致的局部细节缺失。与先前版本相比，面向机器人交互与视频预判的可用性更强，表现出更好的跨任务迁移。
- **你能学到什么**：阅读时可重点关注：为什么让可见token也参与损失能提升几何/运动锚定，而不只是补全被mask区域。第二个要点是分层监督对梯度流和表征分工的影响，尤其是中层特征如何服务密集任务。实践上可借鉴其统一tokenizer与规模化策略，用同一预训练管线覆盖图像和视频，降低多数据形态切换成本。
- **关键洞见**：核心洞见是：视频自监督若想真正服务世界建模，不能只追求全局判别语义，必须把时空局部结构作为一等公民并贯穿网络层级去学习。V-JEPA 2.1 证明了当密集对齐、层级监督与多模态统一训练共同成立时，模型会自然获得“语义一致+时空可定位”的通用表示。这种表示更接近可行动系统所需的感知基础。

### 🎮 4DGS + Diffusion Simulator（1）

#### ⭐ From Prior to Pro: Efficient Skill Mastery via Distribution Contractive RL Finetuning
- **主题**：4DGS + Diffusion Simulator
- **论文链接**：
- **辅助图示**：https://cdn-thumbnails.hf-mirror.com/social-thumbnails/papers/2603.10263.png
![From Prior to Pro: Efficient Skill Mastery via Distribution Contractive RL Finetuning](https://cdn-thumbnails.hf-mirror.com/social-thumbnails/papers/2603.10263.png)
- **这篇在做什么**：可以把方法理解为：先用生成式策略学会“会做很多动作”，再用在线奖励把“做得好的那部分”逐步放大。DICE-RL不是推翻先验重学，而是在原策略附近做有选择的强化，避免训练发散。最终目标是把“广覆盖但一般”的行为先验，变成“成功率高且可落地”的专家策略。
- **核心方法**：输入与表示上，先得到一个基于扩散或流模型的预训练机器人策略，通常以高维视觉观测到动作为映射并覆盖多样行为。关键模块包括残差式离策略RL更新、选择性行为正则（约束别偏离有用先验）以及价值引导动作选择（优先执行高价值候选）。流程上先离线预训练行为先验，再用在线交互反馈进行稳定微调，通过“收缩到高回报区域”完成从prior到pro的升级。
- **和以前有什么不同**：相较传统RL从随机或弱初始化起步，DICE-RL直接继承生成式先验，显著降低探索负担并减少无效试错。相较仅靠行为克隆或静态先验约束的方法，它引入价值驱动的改进机制，能持续提升而非只做保守模仿。相较一些不稳定的在线微调方案，其残差离策略与选择性正则组合更强调稳定、可控地优化。
- **你能学到什么**：阅读时可重点跟踪“分布收缩”在算法中的具体实现：哪些更新在拉近高回报子分布，哪些机制在防止策略坍塌。其次关注残差策略与价值引导如何配合，决定了样本效率和训练稳定性的平衡。若你做真实机器人，建议优先借鉴其“先广覆盖预训练、再小步在线强化”的两阶段范式。
- **关键洞见**：论文最重要的洞见是：预训练生成策略不应只当初始化参数，而应被视为可被RL持续重塑的行为分布。RL微调的本质可表述为对策略分布做“向成功轨迹收缩”，从而在保留多样性的同时提升任务成功率。这一视角把生成建模与控制优化统一到同一分布演化框架中。

### 📌 Other（1）

#### ⭐ Generation Models Know Space: Unleashing Implicit 3D Priors for Scene Understanding
- **主题**：Other
- **论文链接**：
- **辅助图示**：https://cdn-thumbnails.hf-mirror.com/social-thumbnails/papers/2603.19235.png
![Generation Models Know Space: Unleashing Implicit 3D Priors for Scene Understanding](https://cdn-thumbnails.hf-mirror.com/social-thumbnails/papers/2603.19235.png)
- **这篇在做什么**：作者认为，能生成时序一致视频的扩散模型，内部其实已经学会了物体结构、相机变化和部分物理规律。VEGA-3D把这些中间层时空特征“取出来”，再与语言-视觉语义特征融合，让原本偏语义的多模态大模型获得更强的空间判断能力。直观上，就是借生成模型的“世界感”来修复MLLM的“空间盲区”。
- **核心方法**：输入侧同时使用常规视觉-文本语义表示与预训练视频扩散模型在不同噪声步的中间时空表示，后者被视为潜在世界模拟器的几何线索。关键模块是token级自适应门控融合，对每个token动态决定语义特征与生成先验的混合比例，避免简单拼接带来的噪声干扰。训练/推理时无需显式3D监督或改造生成器主体，作为即插即用增强层接入MLLM，在下游3D理解与空间推理任务中直接使用融合表示完成预测。
- **和以前有什么不同**：相较依赖点云、深度、NeRF或外部几何引擎的方法，本文不要求显式3D模态和复杂几何脚手架，减轻了数据与部署负担。相较只在判别模型里强化空间头部的方案，它直接复用大规模生成模型已学到的时空先验，迁移来源更强。其核心增量是“从生成中提取几何”而非“再造几何模块”，路线更轻量也更通用。
- **你能学到什么**：可借鉴点一是把生成模型中间噪声层特征当作通用表征资源，而不是只把生成器用于合成结果。可借鉴点二是token级门控融合思路，适合处理语义强但几何弱、或几何强但语义弱的互补模态。实践中应重点关注抽取层位、噪声步选择与融合稳定性，这些通常决定迁移效果上限。
- **关键洞见**：论文最关键的洞见是：视频生成模型为保证时序一致性，已被迫内化了可迁移的3D结构与物理约束，这些先验可直接服务理解任务。由此，生成模型不只是数据合成器，也可作为“隐式世界模型特征库”。这为构建更懂空间与交互的MLLM提供了一条低监督、可扩展的新范式。
