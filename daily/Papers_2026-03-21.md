### 🧠 World Models（1）

#### V-JEPA 2.1: Unlocking Dense Features in Video Self-Supervised Learning
- **主题**：World Models
- **论文链接**：
- **辅助图示**：https://cdn-thumbnails.hf-mirror.com/social-thumbnails/papers/2603.14482.png
![V-JEPA 2.1: Unlocking Dense Features in Video Self-Supervised Learning](https://cdn-thumbnails.hf-mirror.com/social-thumbnails/papers/2603.14482.png)
- **这篇在做什么**：可以把V-JEPA 2.1理解为“让模型在视频里学会每个位置、每个时刻都说得清楚”的自监督框架。它不是只学一个整段视频的摘要向量，而是让可见和被遮挡区域都参与预测，从而形成更稳定的空间结构和时间一致性。最终得到的特征既能做分类这类全局任务，也更适合做交互预测、深度估计和机器人控制等稠密任务。
- **核心方法**：输入侧同时支持图像与视频token，通过多模态tokenizer映射到统一表示空间；表示学习目标是稠密时空token级特征，而非仅CLS式全局向量。关键模块包括：基于mask的稠密预测损失（可见与遮挡token共同提供监督）和跨多层中间特征的深层自监督头，以强化不同语义层级的可迁移性。训练时在多层同时施加自监督目标并进行大规模数据与模型扩展；推理时直接抽取编码器稠密特征用于下游任务，按任务需要再接轻量读出头或线性探针。
- **和以前有什么不同**：相较早期JEPA或常见视频SSL更偏全局表征的路线，这篇把“稠密可定位、时间连贯”作为核心优化对象，并显式让mask内外token共同学习，减少只靠可见块带来的局部偏置。它还把自监督从最后层扩展到中间层，缓解高层语义过强、低层几何不足的失衡问题。再加上图像/视频统一tokenizer与规模化训练，整体上从单一视频表征器走向更通用的视觉世界模型底座。
- **你能学到什么**：可借鉴点一是目标设计：若下游需要定位、交互或控制，训练时就应让token级时空一致性直接进入损失，而非事后靠微调“补课”。可借鉴点二是监督深度：在多层施加自监督常能提升泛化与线性可读性，尤其对同时覆盖几何与语义的任务有帮助。工程上建议优先验证统一tokenizer与mask策略的兼容性，再逐步放大模型和数据规模。
- **关键洞见**：论文的关键信息是：高质量世界建模不应在“全局语义”和“局部稠密性”之间二选一，二者可以通过合适的自监督目标共同提升。具体地，mask稠密预测+层级监督让表示同时具备语义一致、空间可对齐、时间可延展的特征结构。这解释了它为何能跨动作预测、导航、深度与机器人抓取等任务保持较强迁移性。

### 🗣️ VLA（2）

#### ⭐ FASTER: Rethinking Real-Time Flow VLAs
- **主题**：VLA
- **论文链接**：
- **辅助图示**：https://cdn-thumbnails.hf-mirror.com/social-thumbnails/papers/2603.19199.png
![FASTER: Rethinking Real-Time Flow VLAs](https://cdn-thumbnails.hf-mirror.com/social-thumbnails/papers/2603.19199.png)
- **这篇在做什么**：论文认为，机器人慢并不只是推理慢，而是动作分块策略让“首个有效动作”出现得太晚。它把反应时间拆成TTFA与执行时域共同决定的分布问题，说明传统恒定采样日程会迫使模型先把整段都采完才能动。FASTER的核心是先快速得到近期动作、再细化后续轨迹，从而更快起步且不明显牺牲整体平滑性。
- **核心方法**：输入与表示上，方法仍基于流式VLA的动作chunk生成：由视觉与语言条件生成一段未来动作序列，并在流匹配/去噪采样中逐步得到轨迹。关键模块是Horizon-Aware Schedule，它按时间远近分配采样预算，优先压缩最前几步动作的去噪轮次，使“立即反应段”可极少步数产出。训练范式基本兼容原有flow-based VLA，主要改在推理调度，并结合流式client-server流水线实现边生成边执行，减少有效反应延迟。
- **和以前有什么不同**：以往异步VLA多关注连续性与吞吐，默认恒定采样节奏，隐含前提是整段动作质量优先于首动作时机。本文的增量在于把反应时间形式化分析为由TTFA和执行horizon共同约束，并据此重设计采样日程，而非仅做系统层并行。相比“全段同等精修”的策略，FASTER强调近期动作优先收敛，形成更符合实时交互需求的推理目标。
- **你能学到什么**：做实时机器人策略时，评价指标应从平均成功率扩展到反应延迟分布，尤其关注环境突变后的首动作可用时间。生成式动作模型不必每个时域同等计算，时间感知的预算分配常比盲目加速更有效。算法优化与系统管线要一体化设计：仅改模型或仅改工程都可能达不到端到端实时性提升。
- **关键洞见**：真正限制实时VLA的瓶颈，往往不是总采样步数，而是“首个可执行动作”被整段采样流程阻塞。只要在采样过程中显式区分近端反应与远端规划，就能在几乎不破坏长时域质量的前提下显著提前起动作。FASTER说明了一个重要方向：把动作时域结构直接写进推理调度，比统一日程更贴近机器人在线决策本质。

#### ⭐ Look Before Acting: Enhancing Vision Foundation Representations for Vision-Language-Action Models
- **主题**：VLA
- **论文链接**：
- **辅助图示**：https://cdn-thumbnails.hf-mirror.com/social-thumbnails/papers/2603.15618.png
![Look Before Acting: Enhancing Vision Foundation Representations for Vision-Language-Action Models](https://cdn-thumbnails.hf-mirror.com/social-thumbnails/papers/2603.15618.png)
- **这篇在做什么**：论文核心在于：很多VLA在生成动作时，后层对视觉信息依赖变弱，容易出现“看不清却要动手”的情况。作者让视觉基础模型与VLA主干在更深层共享注意力，把多层视觉特征持续注入动作决策，并用动作引导的剪枝去掉无关视觉token。这样既增强关键视觉线索，又尽量控制计算开销。
- **核心方法**：输入是图像观测与语言指令，先由视觉基础模型产生多层视觉表示，再与VLA主干一起在VL-MoT框架下进行共享注意力交互。关键模块有两部分：一是把视觉专家的多级特征注入VLA深层，缓解深层视觉退化；二是AGVP利用浅层注意力估计与动作相关性，裁剪无关视觉token。训练与推理时都遵循“先保留任务相关视觉，再在深层持续对齐语言与动作”的流程。
- **和以前有什么不同**：相较把LLM/VLA主干当黑盒、仅在输入侧加强视觉的做法，这篇工作直接针对层内信息流做结构性修正，强调深层仍需可用视觉证据。与只做均匀token压缩的方法不同，AGVP由动作相关注意力驱动，目标是“按任务保真”而非单纯降算力。整体上它把可解释分析与架构设计更紧密地结合起来。
- **你能学到什么**：可借鉴的研究方法是先做细粒度诊断：按层分析视觉token对动作生成的影响，再决定该在何处注入视觉信息。工程上可关注“多层视觉注入+任务相关剪枝”的组合：前者提上限，后者控成本。复现时应重点检查剪枝是否误删关键目标区域，以及不同任务下注意力相关性的稳定性。
- **关键洞见**：：VLA性能瓶颈不一定在“有没有视觉编码器”，而在“动作生成深层是否仍保有可用视觉条件”。只在前端增强视觉常被深层语言/动作建模稀释，因此需要跨层、持续的视觉-动作对齐机制。再配合动作引导的选择性视觉保留，才能同时提升精度与效率。

### 🎮 4DGS + Diffusion Simulator（2）

#### ⭐ Matryoshka Gaussian Splatting
- **主题**：4DGS + Diffusion Simulator
- **论文链接**：
- **辅助图示**：https://cdn-thumbnails.hf-mirror.com/social-thumbnails/papers/2603.19234.png
![Matryoshka Gaussian Splatting](https://cdn-thumbnails.hf-mirror.com/social-thumbnails/papers/2603.19234.png)
- **这篇在做什么**：作者希望一个高斯模型像“套娃”一样可截断使用：只取前k个高斯也能成像，k越大质量越好。为此训练时随机抽预算，让模型反复适应不同容量，同时又保持满容量结果不掉线。最终得到的是一个按重要性排序的高斯序列，推理时按算力直接取前缀即可连续调速。
- **核心方法**：输入表示仍是标准3DGS的高斯集合，只是学习出一个有序列表，使前缀子集本身具备可渲染性与逐级增强性。关键模块是随机预算训练：每步采样一个预算k，分别对“前k个高斯渲染结果”和“全量高斯渲染结果”共同优化。训练流程只需两次前向、无需改架构；推理时按目标预算截取前缀渲染，实现连续LoD。
- **和以前有什么不同**：相较离散LoD方法只能提供几个固定工作点，MGS把可用预算从离散点扩展为近连续区间，切换更平滑。相较一些连续LoD方法在满容量会劣化，MGS强调同时优化前缀与全量，目标是保住backbone的上限质量。它的改动位置也更“轻”，主要在训练策略而非新增复杂结构。
- **你能学到什么**：可重点关注“有序性”如何形成：高斯排序策略与预算采样分布会直接影响低预算画质和增长曲线平滑度。阅读时建议把它看成一种多预算一致性训练，与蒸馏或课程学习有相通处但目标不同。复现层面可先在现有3DGS代码中实现双分支损失，再逐步做预算与排序消融。
- **关键洞见**：：LoD不一定要靠多模型或多头结构，也可以通过“前缀可用”的训练目标内生出来。只要让模型在训练期反复面对随机预算，并始终绑定全容量目标，就能兼顾低预算可用性与高预算上限。这样LoD从部署时的工程补丁，变成训练时就被塑形的能力。

#### From Prior to Pro: Efficient Skill Mastery via Distribution Contractive RL Finetuning
- **主题**：4DGS + Diffusion Simulator
- **论文链接**：
- **辅助图示**：https://cdn-thumbnails.hf-mirror.com/social-thumbnails/papers/2603.10263.png
![From Prior to Pro: Efficient Skill Mastery via Distribution Contractive RL Finetuning](https://cdn-thumbnails.hf-mirror.com/social-thumbnails/papers/2603.10263.png)
- **这篇在做什么**：可以把它理解为：先用扩散/流模型学一个“会很多动作”的基础机器人，再用在线奖励把好动作挑出来反复强化。作者把这个过程称为“分布收缩”，即不从零学，而是把原先宽泛行为压到更成功的子分布。这样通常比纯RL更稳，也比只做模仿更能逼近任务最优。
- **核心方法**：输入与表示上，策略以高维像素观测为条件，采用扩散或flow生成动作分布，先得到覆盖面广的行为先验。关键模块是残差式离策略RL微调、选择性行为正则和价值引导动作选择：前者在先验上做小步改进，后两者分别约束别偏离过头与把采样动作朝高价值重排。流程上先离线预训练先验，再在线收集反馈并迭代更新价值与策略，实现从prior到pro。
- **和以前有什么不同**：相较从头离策略RL，它不是直接在全动作空间硬搜，而是依托生成式先验缩小有效搜索区间，因此更稳更省样本。相较纯行为克隆或固定先验方法，它引入在线价值信号持续“提纯”成功轨迹，而非只复现数据均值。相较常见RL微调，它突出“分布收缩”视角和选择性正则+价值引导的组合。
- **你能学到什么**：阅读时可重点抓三点：为何“收缩分布”能同时提升成功率与训练稳定性、选择性正则何时该强何时该弱、价值引导在生成式动作采样中如何落地。复现上建议先做短时程任务验证残差更新是否安全，再扩展到长时程像素操控。评估应同时看最终性能、样本效率和退化/崩溃频次。
- **关键洞见**：最关键洞见是把RL定位为“对已有策略分布做收缩与重加权”的算子，而不是从无到有地构造策略。这样可把预训练模型的广覆盖优势与RL的目标优化能力拼接起来，形成可持续精进的闭环。对机器人而言，这提供了一条从大规模先验到任务级熟练度的实用途径。

#### Generation Models Know Space: Unleashing Implicit 3D Priors for Scene Understanding
- **主题**：Other
- **论文链接**：
- **辅助图示**：https://cdn-thumbnails.hf-mirror.com/social-thumbnails/papers/2603.19235.png
![Generation Models Know Space: Unleashing Implicit 3D Priors for Scene Understanding](https://cdn-thumbnails.hf-mirror.com/social-thumbnails/papers/2603.19235.png)
- **这篇在做什么**：作者认为视频扩散模型要生成时间连贯的画面，必须在内部学到物体结构、相对位置和运动规律，因此这些中间特征可用于“补课”MLLM的空间盲点。VEGA-3D的核心就是把这种隐式3D知识提取出来，再与语义特征按token粒度自适应融合，从而提升场景理解与操作相关推理。
- **核心方法**：输入侧同时使用常规视觉-语言语义表示与预训练视频扩散模型在不同噪声时刻的时空中间特征，后者被视作潜在世界模拟器的几何线索。关键模块是token级自适应门控融合：根据每个token需求动态分配语义信息与几何先验权重，避免粗暴拼接。训练/推理上保持生成模型主体冻结，以插件方式接入MLLM，在下游理解或决策任务中利用融合表示进行答案生成或动作推断。
- **和以前有什么不同**：以往方法多依赖显式深度、点云、NeRF或额外3D监督，常受标注稀缺和跨场景泛化差限制；本文转向利用生成模型隐式先验，减少对专门3D数据管线的依赖。与仅做视觉特征增强的方案相比，VEGA-3D强调从扩散过程不同噪声层抽取动态几何信息，并通过细粒度门控融合提升可用性与稳健性。
- **你能学到什么**：可借鉴的研究方法是“能力重用”：先分析大模型为完成原任务必须具备的隐式能力，再设计轻量接口迁移到新任务，而非从头构建新模态系统。工程上要重视特征提取时机与融合粒度，不同噪声层和token级控制会显著影响空间信息质量。评测上应覆盖场景理解、空间推理与具身操作，验证先验是否真正转化为可执行能力。
- **关键洞见**：：生成模型不仅会“画得像”，还在内部编码了可用于推理的世界结构与物理约束，这些知识可被外接到理解模型中。把视频扩散模型当作潜在世界模拟器，可在不引入重型显式3D监督的前提下补齐MLLM的空间认知短板，为通向物理世界智能提供更可扩展的路线。
